# Finance AI

Десктопное приложение для анализа банковских выписок, учёта транзакций и финансовых целей с советами от облачного ИИ.

- **Скачать (Windows):** [Последний релиз](https://github.com/nikprog1/Finance_AI-2/releases/latest)  
- **Лендинг:** [GitHub Pages](https://nikprog1.github.io/Finance_AI-2/) (включите Pages в настройках репозитория, папка `/docs`).

Подробная инструкция по публикации релизов и настройке GitHub Pages: [docs/PUBLISH.md](docs/PUBLISH.md).  
Замените `REPO_OWNER` на владельца репозитория (например `nikprog1`).
