Finance AI

Анализ банковских выписок и советы по целям

Десктопное приложение для учёта транзакций: импорт CSV-выписок Тинькофф, хранение в SQLite, отчёты по категориям и картам, финансовые цели с прогрессом и рекомендации по сокращению расходов с помощью облачного ИИ (OpenRouter / OpenAI-совместимый API).
